8.33. Colorify

8.33.1. Overview

Figure 16.185. Example for the “Colorify” filter

Original image

Filter “Colorify” applied


This filter renders a greyscaled image like it is seen through a colored glass.

For every pixel, the filter computes a weighted average value of the RGB channels (this is equivalent to desaturating the image based on Luminosity). The resulted color is the product of this average value and the “colorify color”.

Hence, this filter works only on images in RGB mode.

8.33.2. Activate the filter

This filter is found in the image window menu under Colors → Colorify.

8.33.3. Options

Figure 16.186. “Colorify” filter options


A color palette is available containing especially the RGB colors Red, Green, Blue and the CMY colors Cyan, Magenta, Yellow.

You can select your own color by clicking on the Custom Color swatch.
