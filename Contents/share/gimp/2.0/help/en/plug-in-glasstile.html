5.11. Glass Tile

5.11.1. Overview

Figure 17.124.  The same image, before and after applying glass tile effect.

Original image

Filter “Glass Tile” applied


After applying this filter, the active layer or selection is rendered as through a glass brick wall.

5.11.2. Activate the filter

You can find this filter through Filters → Light and Shadow → Glass Tile

5.11.3. Options

Figure 17.125. “Glass Tile” filter options


Preview

If checked, parameter setting results are interactively displayed in preview. Scroll bars allow you to move around the image.

Tile width, Tile length

Sets tile width and length (10-50 pixels).

By default, width and height are linked, indicated by the chain symbol next to the input boxes. If you want to set width and height separately, click on that chain symbol to unlink them.
