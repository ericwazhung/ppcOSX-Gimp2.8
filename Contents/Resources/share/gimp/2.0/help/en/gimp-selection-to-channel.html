4.17. Save to Channel

The Save to Channel command saves the selection as a channel. The channel can then be used as a channel selection mask. You can find more information about them in the Channel Dialog section.

You will find a simple example how to use this command in the introduction of Section 16, “Alpha to Logo Filters”. It shows how to convert a selection to an alpha channel so that you can apply an alpha to logo filter to this selection.

4.17.1. Activate the Command

  • You can access this command from the image menubar through Select → Save to Channel.

  • You can also access it from the Selection Editor.
